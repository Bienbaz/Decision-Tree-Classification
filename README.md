# Decision Tree Classification – Airline Customer Satisfaction

## Project Overview
This project builds an optimized **Decision Tree** classifier to predict passenger satisfaction using survey data. It includes hyperparameter tuning with GridSearchCV, tree visualization, feature importance, and comparison with Logistic Regression.

## Dataset
- **File**: `Invistico_Airline.csv`
- Target: `satisfaction` (binary)
